## üöÄ Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/nithasp/luigi-pizza-new
   ```

2. Navigate to the project directory:

   ```bash
   cd luigi-pizza-new
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm start
   ```

5. Open your browser at:
   ```
   http://localhost:3000
   ```
